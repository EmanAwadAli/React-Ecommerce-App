#  🛒 Ecommerce App
## 📌 Overview
In this project, I created an ecommerce web application using react.

## 📥 Installation

###### Clone Project
```
git clone https://github.com/EmanAwadAli/React-Ecommerce-App.git
```
###### Install npm packages
```
npm i
```
###### Start your Project
```
npm run dev
```
